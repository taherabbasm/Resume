# Resume Website

This is my simple resume website made using only HTML.

It shows my profile, skills, experience, education, and contact details in a clean format.

## Live Website

You can view the website here:

https://taherabbasm.github.io/Resume/

## About This Project

- Built using HTML5 only  
- No CSS  
- No JavaScript  
- Simple structure with headings, lists, and tables  

## How to Use

1. Open the link above.
2. Scroll through the sections to read about my background.
3. All information is displayed on a single page.
